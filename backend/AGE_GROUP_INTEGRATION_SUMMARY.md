## 🔄 Логика работы

### Генерация ситуаций по возрастным группам:

1. **При создании комнаты:**
   - Пользователь указывает возраст при регистрации
   - Система определяет `age_group` по возрасту
   - Комната создается с соответствующим `age_group`

2. **При быстром поиске:**
   - Система находит комнату с подходящим `age_group`
   - Или создает новую комнату с `age_group` пользователя

3. **При генерации ситуаций:**
   - GameService получает `age_group` из комнаты
   - Передает его в AI сервис
   - AI генерирует контент, подходящий для этой возрастной группы

### Возрастные группы и темы:

- **kids (6-12)**: школа, друзья, игры, семья, животные, мультфильмы
- **teens (13-17)**: школа, друзья, соцсети, спорт, музыка, мемы
- **young_adults (18-29)**: учеба, работа, отношения, технологии, путешествия
- **adults (30-59)**: работа, семья, быт, здоровье, карьера
- **seniors (60+)**: семья, здоровье, досуг, воспоминания
