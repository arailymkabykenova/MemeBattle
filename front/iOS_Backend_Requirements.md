### 6.3 События от сервера к клиенту

#### Управление игроками
- **player_joined:** Игрок присоединился к комнате
  ```json
  {
    "player": {
      "id": "string",
      "nickname": "string",
      "avatar_url": "string"
    }
  }
  ```
- **player_left:** Игрок покинул комнату
- **player_ready:** Игрок готов к игре
- **player_unready:** Игрок не готов

#### Игровой процесс
- **game_started:** Игра началась
  ```json
  {
    "rounds_count": "number",
    "time_per_round": "number",
    "players": ["array"]
  }
  ```
- **round_started:** Новый раунд начался
  ```json
  {
    "round_number": "number",
    "situation_text": "string",
    "timer_duration": "number",
    "current_player": "string"
  }
  ```
- **card_played:** Карта сыграна
  ```json
  {
    "player_id": "string",
    "card_id": "string",
    "position": "number"
  }
  ```
- **round_ended:** Раунд завершен
  ```json
  {
    "winner_id": "string",
    "scores": "object",
    "next_round": "number"
  }
  ```
- **game_ended:** Игра завершена
  ```json
  {
    "winner_id": "string",
    "final_scores": "object",
    "rewards": "object"
  }
  ```

#### Системные события
- **error:** Ошибка в игре
- **timer_update:** Обновление таймера
- **connection_lost:** Потеря соединения игрока
- **reconnection:** Восстановление соединения

### 6.4 События от клиента к серверу

#### Управление игрой
- **join_room:** Присоединение к комнате
- **leave_room:** Выход из комнаты
- **set_ready:** Установка готовности
- **start_game:** Запрос на начало игры (для создателя)

#### Игровые действия
- **play_card:** Сыграть карту
  ```json
  {
    "card_id": "string",
    "position": "number"
  }
  ```
- **skip_turn:** Пропустить ход
- **use_ability:** Использовать способность карты

#### Системные
- **ping:** Проверка соединения
- **reconnect:** Восстановление соединения
